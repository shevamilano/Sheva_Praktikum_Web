About Me

Halo, nama saya Baruna Pasha. Saya saat ini sedang menempuh pendidikan di Universitas Pertamina dan mengambil program studi Ilmu Komputer. Ketertarikan saya terhadap dunia teknologi, khususnya dalam pemrograman dan pengembangan perangkat lunak, mendorong saya untuk memilih jurusan ini. Selain belajar tentang teori dan praktik di bidang komputer, saya juga tertarik untuk mengeksplorasi berbagai teknologi baru seperti kecerdasan buatan, pengembangan aplikasi, dan sistem informasi. Saya percaya bahwa teknologi memiliki peran besar dalam membentuk masa depan, dan saya ingin menjadi bagian dari perubahan tersebut. Melalui perjalanan akademik ini, saya berharap dapat mengembangkan kemampuan teknis dan kreatif saya untuk menciptakan solusi inovatif yang berdampak positif pada masyarakat.
